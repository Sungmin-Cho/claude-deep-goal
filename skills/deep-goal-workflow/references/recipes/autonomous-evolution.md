# 레시피: autonomous-evolution

deep-evolve의 자율 실험 루프를 fitness metric 목표치까지 반복 실행하는 goal이다. 평가자가 매 턴 후 metric이 목표치에 도달했는지 판정할 수 있도록 metric 값을 대화에 명시 보고하는 것이 핵심이다.

---

## 트리거 (감지 조건)

`deep-evolve` 플러그인이 감지될 때 이 레시피를 제안한다.

- `deep-evolve` — 자율 실험·변이·선택 루프 워크플로우

`deep-evolve`가 없으면 단발 goal 폴백을 사용한다.

---

## 전제 (필요 플러그인)

| 플러그인 | 역할 |
|---|---|
| `deep-evolve` | 자율 실험 루프: 후보 생성 → 측정 → 선택 반복 |

---

## 시퀀스

```
[1] deep-evolve 초기 상태 설정 (기준 fitness metric 측정)
[2] 실험 루프 반복:
    a. 후보(변이/탐색) 생성
    b. fitness metric 측정
    c. 측정값 대화에 보고
    d. 목표치 도달 판정 → 도달 시 종료
[3] 상한(턴/이터레이션) 도달 시 현재 최고 결과와 함께 종료
```

---

## 종료조건

다음 중 **하나**가 충족되면 완료:

1. **fitness metric ≥ 목표치** — 측정값이 대화에 보고된 상태
2. **상한 도달** — 설정된 최대 이터레이션 또는 턴 수 초과

종료 시 현재 최고 fitness metric 값과 달성 여부를 대화에 명시 보고한다.

---

## 컴파일된 `/goal` 예시

### Claude (fitness metric 표면화 포함)

```
deep-evolve 루프로 <목표 시스템>의 <fitness metric>을 <목표치>까지 향상시킨다.
매 이터레이션마다 측정한 <fitness metric> 값을 대화에 보고한다.
종료조건: <fitness metric> ≥ <목표치>가 대화에 보고됨, OR 이터레이션 <N>회 상한 도달.
불변 제약: <변경 금지 항목 — 예: 인터페이스 시그니처 유지, 기존 테스트 삭제 금지>.
현재까지 최고 metric 값과 달성 여부를 종료 시 대화에 요약 보고할 것.
or stop after 50 turns.
```

### Codex (contract 형태)

```
/goal 목표: deep-evolve로 <목표 시스템>의 <fitness metric>을 <목표치> 이상으로 향상시킨다.

달성 조건:
- <fitness metric> ≥ <목표치> (대화에 측정값 보고됨)
- 또는 이터레이션 <N>회 상한 도달 시 최고값 보고 후 종료

변경 금지: <불변 제약>
검증: <fitness metric 측정 커맨드>

각 이터레이션마다 측정값을 진행 로그에 기록.
종료 시 최고 metric 값과 달성 여부를 요약 보고.
```

---

## 주의 (현실적 제약)

### deep-evolve 자체 루프와 goal 상한의 관계

deep-evolve는 자체 루프를 가진다. goal의 `or stop after N turns` 상한이 deep-evolve의 예상 이터레이션보다 작으면 루프 중간에 goal이 종료될 수 있다. 이터레이션당 예상 턴 수를 추산해 여유 있게 상한을 설정한다.

### fitness metric 정의 명확화

`fitness metric`이 모호하면 재구성 대화(3단계)에서 구체화한다. 예:
- "정확도 95% 이상" → `pytest --cov` 결과로 측정 가능 여부 확인
- "응답 속도 100ms 이하" → 벤치마크 커맨드 식별

metric이 자동 측정 불가능한 주관 지표이면 반려 후 재구성을 권고한다.

### 평가자 표면화 없으면 종료 판정 불가

Claude 평가자는 도구 없이 대화 출력만 판정한다. 매 이터레이션마다 측정값을 대화에 보고해야 평가자가 "목표치 도달 여부"를 판정할 수 있다. `<fitness metric> ≥ <목표치>가 대화에 보고됨`을 종료조건에 포함하는 이유다.

### 목표 미달 시 처리

상한에 도달했지만 목표치 미달인 경우:
- 최고 metric 값과 현황을 대화에 보고하고 종료한다.
- 계속 진행이 필요하면 새 goal을 설정하거나 목표치를 조정한다.
